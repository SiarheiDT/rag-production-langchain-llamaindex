# Part 1 — LangChain Basic Concepts Recap

This folder contains code files created strictly from the direct code examples
shown in the lesson **"1.1 LangChain: Basic Concepts Recap"**.

Included examples:
- `m_01_001_csv_loader.py`
- `m_01_002_wikipedia_loader.py`
- `m_01_003_chat_openai_basic.py`
- `m_01_004_openai_embeddings.py`
- `m_01_005_llm_chain_rainbow.py`
- `m_01_006_lcel_rainbow.py`
- `m_01_007_sequential_chain_social_post_review.py`

Notes:
- These files follow the lesson examples only.
- Conceptual sections without runnable code in the lesson were not expanded into extra scripts.
- OpenAI-based examples require `OPENAI_API_KEY`.

Additional setup:
- All scripts now load environment variables from the project root `.env` file via `common/env_loader.py`.
- Install `python-dotenv`.
